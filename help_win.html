QR Tools

1. By default, this extension displays the QR code corresponding to the current web page URL;

2. You can modify the URL string in the input box, and the QR code will automatically update.

• Chrome on iPhone and iPad cannot install extensions, you need to use Safari.

• There is a Safari version of this extension. Search “QR Tools - Customizable” in App Store and install it, then you can use it on your iOS and macOS devices.
